## 项目简介

本项目实现了基于注意力U-Net的皮肤病变分割算法，并在ISIC数据集上进行了训练和评估。同时实现了Otsu、HSV分割、K-means、分水岭等多种经典图像处理方法进行横向对比。

**核心工作：**
- ✅ 实现了标准U-Net和引入CBAM注意力的U-Net
- ✅ 实现了6种经典图像分割方法用于对比
- ✅ 在ISIC 2018数据集上进行了完整训练和评估
- ✅ 完成了深度学习与经典方法的横向对比分析
- ✅ 提供了完整的可视化结果和失败案例分析

## 研究背景与意义

### 临床意义

皮肤癌是全球最常见的恶性肿瘤之一，其中黑色素瘤虽然仅占皮肤癌病例的5%左右，但其致死率极高。早期发现和治疗可显著提高患者生存率（晚期5年生存率不足15%，早期可达95%以上）。

皮肤病变图像分割的核心价值：
- 辅助医生快速定位病灶区域
- 定量评估病变大小、形状、边界不规则程度
- 为ABCD规则（Asymmetry, Border, Color, Diameter）提供量化依据

### 技术挑战

1. **细小病变漏检**：早期病灶面积小，容易误判为背景
2. **边界模糊**：病变边缘常与正常皮肤渐变过渡
3. **类间相似性高**：恶性与良性病变视觉差异微妙
4. **数据不平衡**：恶性样本占比低
5. **跨数据集泛化**：不同拍摄设备、光照条件影响大

使用**ISIC 2018**皮肤病变数据集。

- 图像尺寸统一缩放至256×256像素
- 像素值归一化到0-1区间
- 数据增强：随机旋转、翻转、平移缩放、亮度对比度调整、噪声添加

## 方法介绍

### 1. 经典图像处理方法

| 方法 | 原理简介 |
|------|----------|
| **Otsu** | 基于灰度直方图类间方差最大化自动寻找阈值 |
| **Multi-Otsu** | 多阈值Otsu，将图像分为多个灰度等级 |
| **HSV分割** | 基于色调和饱和度信息提取病变区域 |
| **K-means** | 聚类算法，将像素分为病变和背景两类 |
| **分水岭** | 基于梯度信息的区域分割算法 |
| **组合方法** | 多数投票机制，结合Otsu、HSV、K-means |

### 2. 深度学习方法

#### U-Net架构

U-Net由编码器（下采样）和解码器（上采样）组成，通过跳跃连接保留空间细节信息。

#### CBAM注意力机制

CBAM包含两个子模块：
- **通道注意力**：学习每个特征通道的重要性权重
- **空间注意力**：学习每个空间位置的重要性权重

#### 注意力U-Net

将CBAM模块嵌入到U-Net的跳跃连接处，使网络更加聚焦于病变区域。

#### 组合损失函数

```
Loss = BCE Loss + Dice Loss
```

BCE Loss提供稳定梯度，Dice Loss直接优化区域重叠度。

## 结果分析

### 深度学习 vs 经典方法

注意力U-Net的Dice系数（0.8647）显著优于最佳经典方法HSV（0.5754），提升约50%。主要原因：

1. **层次化特征学习**：深度学习自动提取从边缘纹理到语义的多层次特征
2. **端到端优化**：所有参数共同优化以最小化分割损失
3. **注意力机制**：CBAM模块使网络聚焦于病变区域
4. **非线性建模**：深度网络能够拟合复杂的决策边界

### 各经典方法对比

- **HSV和K-means**效果相对较好（Dice约0.57），利用颜色信息有效
- **Otsu和分水岭**效果较差，因依赖灰度对比度和清晰边缘
- **组合方法**相比单一经典方法有提升，但仍远不如深度学习方法

### 模型稳定性

注意力U-Net的标准差（0.1659）虽然随Dice提升而增大，但对难例的鲁棒性仍有提升空间。主要困难来源于低对比度病变和极小病灶。

## 失败案例分析

### 典型失败样本

测试集中发现以下类型样本分割效果较差：

1. **低对比度病变**：病变与周围皮肤颜色差异极小
2. **被遮挡的病变**：毛发覆盖导致假阳性
3. **极小病变**：面积小于图像1%时容易漏检

### 改进方向

1. 使用更高分辨率输入（512×512）保留细节
2. 增加低对比度病变的数据增强
3. 引入边界感知损失函数
4. 使用模型集成提升稳定性
